# Conversão de Imagem para Tons de Cinza e Binário (Preto e Branco)

Este projeto converte uma imagem original em dois formatos:
1. **Imagem em Tons de Cinza**: Onde cada pixel tem a mesma intensidade de R, G e B (escala de cinza).
2. **Imagem Binária (Preto e Branco)**: Onde cada pixel é convertido em preto ou branco, dependendo de um limiar de intensidade de cinza.

## Requisitos
- Python 3.x
- Pillow (PIL) para manipulação de imagens

### Como Instalar a Biblioteca Pillow
